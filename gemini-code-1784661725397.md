# NexusDoc Studio | Modern Visual Document Automation

A modern, high-performance, 100% client-side Document Processing Studio and Infinite Canvas Workflow built with **Tailwind CSS**, **Lucide Icons**, and **pdf-lib**.

## 🌟 Highlights & UI/UX Features
- **Dual Studio Mode**: Switch between a streamlined **Quick Studio** and a visual node-based **Canvas Workspace**.
- **Interactive Page Grid**: View rendered thumbnails of multi-page PDFs directly in your browser.
- **Client-Side Compression & Optimization**: Converts heavy images to next-gen WebP format and optimizes PDF streams without sending data to an external server.
- **Dynamic Watermarking**: Add custom watermark stamps on the fly.
- **Dark/Light Glassmorphism Theme**: Clean, responsive interface built with Tailwind CSS.

## 🚀 Hosting on GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Set the source branch to `main` (or `master`) and folder to `/ (root)`.
4. Your site will be published at `https://<username>.github.io/<repo-name>/`.
